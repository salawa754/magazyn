# magazyn referencji
## Lekka i intuicyjna aplikacja webowa sluzaca do zarzadzania baza referencji przypisanych do konkretnych półek magazynowych.
Projekt zostal stworzony z mysla o szybkosci wyszukiwania i estetyczntm czytelnym interfejsie uzytkownika.
Część referencji, w firmie ktorej pracowalem, nie miala przypisanych lokalizacji a pracownicy odpowiedzialni za ich przypisanie nie chcieli tego zrobic, przez co trudno bylo je znalezc. Stworzylem wiec ten projekt by ulatwic oraz przyspieszyc prace.
## Główne funkcje:
- **Błyskawiczne wyszukiwanie**
- **Interaktywny interfejs**
- **filtrowanie lokalne**
- **responsywność**
## Technologie:
- **HTML5**
- **CSS3**
- **JavaScript**
